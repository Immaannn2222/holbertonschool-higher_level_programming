Using ``MyList``
----------------------

>>> MyList = __import__('1-my_list').MyList

Now use it:

Subtle:
>>> my_list = MyList()
>>> my_list.append(6)
>>> my_list.append(120)
>>> my_list.append(0)

>>> print(my_list)
[6, 120, 0]

>>> my_list.print_sorted()
[0, 6, 120]

Negative Numbers:
>>> my_list = MyList()
>>> my_list.append(-8)
>>> my_list.append(-2)
>>> my_list.append(-14)
>>> my_list.print_sorted()
[-14, -8, -2]

One element:
>>> my_list = MyList()
>>> my_list.append(3)
>>> print(my_list)
[3]
>>> my_list.print_sorted()
[3]

None:
>>> my_list = MyList(None)
Traceback (most recent call last):
...
TypeError: 'NoneType' object is not iterable
